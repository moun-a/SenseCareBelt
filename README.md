# SenseCare Belt

Smart wearable safety belt for real-time fall detection and health monitoring.

## Project Overview
SenseCare Belt is an IoT-based wearable system designed to detect falls and monitor vital signs in real-time.  
The system communicates alerts wirelessly and can trigger an external alarm system.

## Features
- Fall detection using MPU6050
- Heart rate monitoring
- GPS location tracking
- LoRa communication
- Emergency alarm system
